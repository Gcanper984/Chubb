PASOS A SEGUIR

PASO 1

Para intalar Recuva, tendremos que visitar la página web de Cleaner y buscaremos en ella la aplicación que deseamos instalar

PASO 2

Para la recuperación, la mejor opción será Recuva

PASO 3

Una vez empezada la descarga, esperamemos a que acabe

PASO 4

Cuando termine la instalación anterior nos aparecerá esta ventena. Pulsaremos en INSTALAR

PASO 5

Cuando se instale, en la siguiente pestaña pulsaremos en Run Recuve

PASO 6

Cuando pulsamos en Run Recuve nos aparecerá la siguiente pestaña, pulsaremos en Next

PASO 8

La aplicación empezará a escanear nuestro equipo

PASO 9

Una vez terminado el escáner, ya tendriamos instalado Run Recuve
